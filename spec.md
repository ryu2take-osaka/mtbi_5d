# Project: MBTI 5D Visualizer (Refactored Full Spec)

## 1. 概要

MBTIの5指標（E-I, S-N, T-F, J-P, A-T）を、3次元空間および「認知機能スタック」の連動によって可視化するWebエンジン。
単なる相性診断に留まらず、個々の認知機能が「A/T（安定/揺れ）」や「LV（使用頻度）」によってどのように振る舞うかを詳細に解き明かす。

---

## 2. ユーザーインターフェース (UI/UX)

### 2.1 動作モード
1. **EDITING (編集)**: スライダー操作中。球体のみ即時更新。
2. **PLAYING (再生)**: 解析完了後、以下のいずれかのモードで動作：
   - **Compatibility (相性モード)**: 20秒サイクルで相性と補完点を解説。
   - **Individual Description (個別解説モード)**: 選択したユーザーの各認知機能を解説し、次に**スタック特性（Expanded/Inverted/Tilted/Balanced）の総評**を行い、最後に「二つ名（Epithet）」と「通称」を公開。

### 3.1 称号（二つ名）決定ロジック
個別の機能解説の最後に、タイプとA/T、および機能の使い方の偏り（Inverted）に基づいて以下の称号を授与する。

- `S/I`: 第3機能(LV3)が第2機能(LV2)より強い場合は `I`（潜在的）、そうでない場合は `S`（通常）。
- `A/T`: 自己主張型は `A`、激動型は `T`。

### 3.2 スタック特性（総評）ロジック
個別の機能解説の後に、全機能のバランスから以下の4つの状態を判定する。

| 状態 (Key) | 判定条件 | 意味 |
| :--- | :--- | :--- |
| **expanded** (拡張型) | `f4 >= f2 * 0.75 && f4 >= 20` | 劣等機能の活性・成長期 |
| **inverted** (反転型) | `f3 > f2 + 5` | Loop傾向・内的/外的への沈潜 |
| **tilted** (傾斜型) | `Math.abs(f2 - f3) <= 5` | 2nd/3rdの拮抗・多面的な個性 |
| **balanced** (均衡型) | その他 | 標準的・安定した精神構造 |

## 4. 演出とビジュアル (Visual Effects)

### 4.1 輝度演出 (Glow & Bloom)
- **カラーグループ**: NT（紫）、NF（緑）、SP（黄）、SJ（青）。
- **REVEAL演出**: 個別解説の終盤（18秒目〜）で、解説対象となっているユーザーの球体のみを通常の3倍に発光（emissiveIntensity）ブースト。これにより、どっちの診断結果が表示されているかを視覚的に明示する。NT型（紫）は鮮やかなバイオレット（#bf5af2）を採用。

### 4.2 ナレーション表示
- **同期制御**: `requestAnimationFrame` 同期により、ブラウザ間の描画ラグを解消。
- **高視認性デザイン**: 影（text-shadow）を二重に配置し、背景の3Dオブジェクトと重なっても文字が埋もれないように設計。

## 5. 技術仕様

### 5.1 モジュール構成
- `js/score.js`: 補完点（Rescue）と鏡像係数（Mirror Factor）の計算。
- `js/render.js`: Three.jsによるWebGLレンダリング。
- `js/narration.js`: 称号決定ロジックとタイムライン制御。

### 5.2 外部依存
- **Three.js**: unpkg CDN経由で `importmap` を用いてロード。

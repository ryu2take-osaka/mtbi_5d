# Project: MBTI 5D Visualizer (Interactive Engine)

## 1. 概要

MBTIの5指標（E-I, S-N, T-F, J-P, A-T）を  
3次元空間・色・発光によって直感的に可視化するWebアプリケーション。

本ツールは「性格＝エネルギーの振る舞い」として表現し、  
ユーザーが操作を通じて性格差や相性を“体験的に理解する”ことを目的とする。

---

## 2. コアコンセプト

- 性格は静的なラベルではなく「動的なエネルギー」
- 数値ではなく「視覚的現象」として理解させる
- UI説明を最小化し、操作と反応で理解させる

---

## 3. データ構造

### 入力形式（UI）

各指標は 0〜100 の連続値とする

| 指標 | 意味 |
|------|------|
| E-I | 外向性 |
| S-N | 感覚 / 直観 |
| T-F | 思考 / 感情 |
| J-P | 判断 / 知覚 |
| A-T | 自己主張 / 慎重 |

---

### 内部変換

```js
normalized = (value - 50) / 50  // -1.0 〜 +1.0
````

---

## 4. ビジュアルマッピング

### 空間（構造）

| 指標  | 要素 |
| --- | -- |
| S-N | X軸 |
| T-F | Y軸 |
| J-P | Z軸 |

---

### 質感（エネルギー）

| 指標  | 要素      |
| --- | ------- |
| E-I | 発光強度    |
| A-T | サイズ（密度） |

## 5. グループカラー

MBTIタイプを4グループに分類し、ベースカラーとする

| グループ | 条件    | 色          |
| ---- | ----- | ---------- |
| NT   | N + T | #9b59b6（紫） |
| NF   | N + F | #2ecc71（緑） |
| SJ   | S + J | #3498db（青） |
| SP   | S + P | #f1c40f（黄） |

---

## 6. 発光表現

### 基本仕様

* E（外向）に寄るほど発光強度が増加
* 発光は以下の3層で構成

#### ① ベース発光

色の明度として反映

#### ② Emissive

マテリアルの自己発光

#### ③ Bloom（ポストエフェクト）

画面全体に拡散する光

---

### 発光制御

```js
glow = EI / 100
emissiveIntensity = 0.3 + glow * 2.0
```

---

## 7. 2人比較モード

### 構成

* 2つの独立した性格データ（A / B）
* 同一空間に2つの球体を配置
* リアルタイムで相性を更新

---

### UI仕様

* タブ切替（A / B）
* 同時編集は行わない
* スマホでは単一操作に限定

---

## 8. 相性ロジック

### Core Distance

```js
D = sqrt(dx² + dy² + dz²)
```

---

### スコア算出

```js
score = 1 - (D / 3)
```

---

### ランク

| スコア     | 評価     |
| ------- | ------ |
| 0.8〜1.0 | S（共鳴）  |
| 0.6〜0.8 | A（高相性） |
| 0.4〜0.6 | B（中間）  |
| 0.2〜0.4 | C（差異）  |
| 0.0〜0.2 | D（対立）  |

---

## 9. 関係性の可視化

### Link（接続線）

* 2点間を線で接続
* 距離に応じて透明度変化

```js
opacity = 1 - distance * k
```

## 10. インタラクション

### 操作

* タッチドラッグ → 回転

* スライダー縦並び

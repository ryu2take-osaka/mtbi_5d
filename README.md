# MBTI 5D Visualizer

## コンセプト
> **「性格を静的なラベルから、空間に浮かぶ光り輝く動的な存在、そして時間を通じて理解される物語へと昇華させるエンジン」**

MBTIの5指標（E-I, S-N, T-F, J-P, A-T）を、単なる4文字の記号としてではなく、3次元空間の座標、エネルギーの輝き、そして認知機能の「水位」として可視化します。

## 主な機能
- **3D Spectrum Mapping**: 性格特性をX/Y/Z軸の空間配置で直感的に把握。
- **Cognitive Stack Visualization**: Harold Grant理論に基づく8つの認知機能の強さを「水位表現」で可視化。
- **Dynamic Narrative Engine**: あなたの精神性が時間とともに解き明かされる40秒間のナレーション演出。
- **Compatibility Analysis (Mirror Factor)**: 二人の機能の噛み合わせを「救済（Rescue）」と「鏡像（Mirror）」の観点からスコアリング。
- **Epithet System**: 64種類のユニークな「二つ名」を授与。

## 使い方
1. **あなた/あいてタブ**でスライダーを動かし、自分や相手のMBTI（16Personalities準拠）を入力します。
2. その場で認知機能の強さや説明がリアルタイムに更新されます。
3. **SCOREタブ**をクリックすると、二人の相性解析と補完ビームの演出が始まります。
4. 解説が終わる頃には、二人の関係性に新しい名前がついているはずです。

## 技術スタック
- Three.js (WebGL)
- Vanilla JS (ES Modules)
- Modern CSS (Grid/Flexbox/CSS Variables)
- CDN-driven (Zero build step)

### スタック総評(オリジナル要素)

| 状態 | 成長フェーズ | 意味 |
| :--- | :--- |:---|
| **expanded** (拡張) | 進化・変革期| 劣等機能が活性化している成長期 |
